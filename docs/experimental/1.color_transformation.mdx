# 颜色转换器

在 Unity 中，Color 对象使用的是 RGB 空间。但在进行颜色补间插值时，RGB 空间并不能很好地反映人眼对颜色变化的感知，
可能会导致过渡不自然或感官亮度波动等问题。严重影响动画的展示效果。

为了解决这一问题，我们提供了基于 [OKLCH](https://oklch.com) 颜色空间的补间实现，该空间在色彩过渡方面更符合人类视觉感知，
在整体视觉效果上更流畅。

实现：[ColorTransformation](https://github.com/MorizeroDev/Milease/blob/master/Scripts/Milease/Translate/ColorTransformation.cs)
